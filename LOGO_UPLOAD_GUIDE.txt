üéØ HOW IT WORKS:
---------------

FOR CUSTOMERS:
1. Add products to cart
2. Go to cart page
3. See purple box below each product: "Add Your Logo (Optional)"
4. Click or drag & drop logo file
5. Preview appears instantly
6. Can remove and re-upload if needed
7. Proceed to checkout (logo is saved automatically)

FOR ADMIN:
1. Customer places order with logo
2. Logo is saved with the order
3. Go to Admin ‚Üí Manage Orders
4. Click "Download Print File" button
5. System automatically:
   - Loads t-shirt image
   - Overlays customer logo on chest
   - Merges into single high-res image
   - Downloads as PNG file
6. Use this file for printing!

üîß TROUBLESHOOTING:

Q: Logo not uploading?

Q: Logo preview not showing?

Q: Print file not downloading?
A: Make sure browser allows downloads, try different browser

Q: Logo position wrong?
A: Contact developer to adjust position in code
